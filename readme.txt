Prioritize gettext entries based on UI elements usage.

== Description ==

Nimrod is a web-based internationalizaton tool that allows you to prioritize UI strings in many different ways. 
In order to make the most of its potential, we have integrated the tool as a Wordpress plugin.

Please refer to the followinf research paper for technical info:
* Leiva, L. A. and Alabau, V. On String Prioritization in Web-based User Interface Localization. In Proc. WISE, 2014.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
